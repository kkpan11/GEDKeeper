Atajos de teclado

A continuación se muestra la lista de atajos de teclado disponibles en GEDKeeper:

Ventana principal
[Ctrl + N] Crea una nueva base de datos.
[Ctrl + O] Abre una base de datos existente.
[Ctrl + X] Abandona la aplicación.
[Ctrl + F11] Abre la ventana de scripts
[F1] Muestra el contendio de ayuda(este documento).
Ventana de base de datos, tablas de registros
[Ctrl + S] Guarda la base de datos actual.
[Ctrl + I] Añade un nuevo registro.
[Ctrl + Enter] Modifica el registro selecionado.
[Ctrl + L] Borra el registro seleccionado.
[Ctrl + A] Visualiza el arbol de antecesores.
[Ctrl + D] Visualiza el arbol de descendientes.
[Ctrl + P] Crea un pedigrí de acuerdo al esquema de d'Aboville.
[Ctrl + K] Crea un pedigrí de acuerdo al esquema de Konovalov.
[Ctrl + M] Abre la ventana de mapas
[Ctrl + T] Abre la ventana de estadísticas
[Ctrl + F] Abre la ventana de filtrado de datos
Ventana de diagramas de árbol
[F5] Recontruye el árbol.
[F6] Hace que el registro de la persona seleccionada sea el punto de partida y reconstruya los antepasados y descendientes de la persona seleccionada.
[F7] Si se activa la opción de mostrar grados de relación, se reconstruirán los grados de relación de la persona seleccionada.
