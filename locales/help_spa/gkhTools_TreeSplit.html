Herramientas: División de base de datos

Normalmente, divides tu base de datos cuando necesitas eliminar por completo a algunas personas o familias de la base de datos, necesitas hacer una copia de esas personas o familias para trasladar los datos a otra base de datos o necesitas enviarlos a otros investigadores.

Antes de ejecutar esta herramienta, debes seleccionar la persona que deseas copiar en la lista de personas de la pestaña "Individuos". A continuación, utiliza uno de los cuatro métodos disponibles en esta herramienta: seleccionar todas las relaciones, seleccionar sólo los antepasados, seleccionar sólo los descendientes o seleccionar sólo la familia. Una vez seleccionadas todas las personas, puede guardarlas en un archivo de base de datos independiente o eliminarlas todas de la base de datos actual. Puedes combinar estas operaciones— es decir, guardar primero los resultados de la selección en una base de datos y, a continuación, eliminarlos de la base de datos actual con el botón "Eliminar".

Al utilizar esta herramienta se muestran dos listas: la lista de la izquierda contiene todas las personas seleccionadas, y la lista de la derecha contiene las personas que no han sido seleccionadas por la herramienta.
