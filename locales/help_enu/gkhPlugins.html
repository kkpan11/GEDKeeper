Plugins

There are some plugin modules available with the default installation:

  • Expression calculator, a standard service plugin.
  • Calendar, a standard service plugin.
  • Names reference book, a standard service plugin.
  • Time line, a standard service plugin.
  • Streamed data input plugin streamlines mass data input from sources.
  • Pedigree import plugin adds data from an external text pedigree.
  • Full-text search in the database, works alike a Web search engine.
  • "Conway Life" game plugin helps you to relax.
  • Chronicle of events.
  • Cloud of names and facts.
  • Set of standard reports (ancestor statistics, name frequency, personal events, name phonetics, list of places, list of archives and sources).
  • GEDmill, website generator.
  • Navigator.

Note: for self-localization of plugins, localization (translation) files for all plugins are located in the `plugins` folder. The files have the same name as the plugin name, the extension is a three-letter language code.

See also: Expandability.
