Диалог выбора записи

Диалог выбора записи - специальный диалог, используемый в большом числе различных мест в программе. Каждый раз, когда при редактировании одной записи, нужно выбрать или добавить какую-либо другую - появляется этот диалог. У него три функции: выбрать имеющуюся запись определенного вида, отфильтровать записи по фрагменту названия, или добавить запись, если нет подходящей.

В каждый момент вызова этого диалога, он содержит список записей в зависимости от требуемого вида. Если пользователь добавляет в персональную запись одного из родителей или ребенка - этот диалог содержит список всех персон в базе данных. Если к источнику присоединяется архив - этот диалог содержит список архивов. Если в диалоге редактирования событий добавляется какое-то место - этот диалог содержит список всех мест в справочнике базы данных.

Например: если пользователь редактирует человека и необходимо присоединить одного из родителей - появляется этот диалог. В первую очередь предполагается, что родитель уже мог быть введен ранее и можно попробовать его отыскать с помощью фильтра (см. Маски фильтрации). Если пользователь точно знает, что такого родителя в базе данных еще нет - можно сразу нажать на кнопку "Добавить".
