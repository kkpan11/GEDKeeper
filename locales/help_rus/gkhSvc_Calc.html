Калькулятор выражений

Простой в использовании калькулятор, позволяющий выполнять расчет выражений с использованием основных математических операций и временных переменных.

Такой калькулятор удобен при поточной обработке источников типа ревизских сказок или исповедальных книг. Калькулятор может быть открыт одновременно с любыми другими диалогами. Между полем результата расчета и полями ввода дат в диалоге правки событий/фактов поддерживается перетаскивание. Т.е. в процессе обработки источника можно в калькулятор вводить выражения вида "1896-56" (год источника - возраст персоны = год рождения), нажать [Enter] для расчета, после чего полученный результат перетаскивать мышкой в поле даты (результат интерпретируется как год).

Если установить галочку в поле "Поместить результат в буфер обмена", то результат расчета будет автоматически копироваться в буфер обмена для ручной вставки в нужном месте.

Если установить галочку в поле "Расчет года событий", то при одновременно открытом диалоге событий результат будет автоматически копироваться в поле даты.
