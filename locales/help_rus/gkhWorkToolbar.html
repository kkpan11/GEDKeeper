Панель инструментов

Основная панель инструментов программы выглядит следующим образом:

Назначение кнопок на ней следующее:

Управление базами данных
 Создать новую базу данных
 Загрузить существующую базу данных
 Сохранить изменения в базу данных
Управление записями (в зависимости от открытой закладки)
 Добавить новую запись
 Редактировать запись
 Удалить запись
 Фильтрация записей
Родословные деревья и росписи (только закладка персон)
 Открыть древо предков выбранного человека
 Открыть древо потомков выбранного человека
 Открыть совмещенное древо предков и потомков выбранного человека
 Сформировать родословную роспись для выбранного человека
Разное
 Сформировать статистику по записям людей
 Отправить файл по почте
 Частное представление
Навигация
 При навигации - возврат на предыдущую запись
 При навигации - возврат на следующую запись
