Виды записей базы данных

Программа GEDKeeper позволяет вводить и редактировать в базах данных следующие виды записей:

  • Персональные записи
  • Семьи
  • Заметки (текстовые заметки к различным записям базы данных, фрагменты документов, комментарии)
  • Мультимедиа-материалы (ссылки на документы, фотографии, аудио- и видео-записи)
  • Источники (частные документы и дела архивных фондов)
  • Архивы (частные архивы информации и государственные организации)
  • Группы (социальные объединения людей по некоторому признаку)
  • Исследования (основные направления поиска и их приоритеты)
  • Задачи (отдельные задачи исследований)
  • Коммуникации и корреспонденция (информация о переписке и контактах с родственниками и другими исследователями, архивными организациями)
  • Места (географические места и их координаты)

Для каждого вида записей выведена отдельная закладка основного окна программы:

С записями любого вида доступны следующие общие операции: добавление, редактирование, удаление, фильтрация.

Каждая закладка записей делится на две области: слева таблица с перечнем записей, справа - сводка с информацией по отдельной выбранной строке.

Каждый параметр записи, выведенный в виде столбца таблицы, может быть отсортирован по возрастанию или убыванию при помощи клика мышкой на заголовке столбца. Таблицы с перечнем записей не допускают прямого редактирования данных непосредственно в таблице для исключения случайных изменений. Для перехода в диалог редактирования необходимо дважды кликнуть по нужной строке, либо на выделенной нажать клавиши [ Ctrl + Enter ].

Каждый тип записи имеет два основных независимых параметра: идентификатор записи и дата/время последнего изменения (столбцы таблиц "№" и "Изменено"). Остальные параметры могут отличаться для каждого типа записи.

Данные таблиц можно отфильтровать нажатием кнопки на панели инструментов или выбрать пункт меню "Правка\Фильтр", или нажатием клавиш [ Ctrl + F ]. На закладке персональных записей также действует функция быстрого поиска, пункт меню "Правка\Поиск".
